En este trabajo se va a trabajar en una base de datos de calificaciones sacada del Student Performance Data Set en el UCI Machine Learning Repository para intentar predecir la calificacin de un alumno dadas las siguientes variables:

Escuela. Indica si el estudiante en cuestin asista a la escuela Gabriel Pereira (GP) oa la escuela Mousinho da Silveira (MS).

Sexo. F para mujeres y H para hombres.

Edad. Edad del estudiante, en aos.

HorasDeEstudio. Cantidad de horas de estudio: 1 indica menos de dos horas, 2 indica de dos a cinco horas, 3 indica de cinco a diez horas, 4 indica ms de diez horas.

Reprobadas. Indica la cantidad de materias reprobadas previamente.

Internet. Si el estudiante tena acceso (yes) o no (no) a internet en su casa.

Faltas. Cantidad de veces que falt a clases.

G1. Calificacin del primer periodo, escala del 0 al 20.

G2. Calificacin del segundo periodo, escala del 0 al 20.

G3. Calificacin final, escala del 0 al 20

Esto quiere decir que tienen texto, o una combinacin de caracteres y letras. Hace sentido dentro del contexto de la base de datos, porque son las que tienen caractersticas cualitativas y categricas. Podemos cambiar estas variables por variables tipo dummy, para poder trabajar con ellas.

Haremos uso de un ciclo for que pase por todos los ndices que corresponden a las variables del tipo object. Para encontrar esos indices usaremos la funcin where de la libreria numpy usando como argumento df.dtypes==object.

Dentro del ciclo for vamos a usar la funcin get dummies de pandas. En esta vamos a usar como argumento las columnas correspondientes a las variables que queremos cambiar, el termino drop_first = True para hacer que solo se generen l-1 variables (una es redundante, por lo que la eliminamos), y dtype = 'float64', esto porque por predeterminado se pueden llegar a generar variables booleanas (0,1) en lugar de enteros, y las bool pueden no ser compatibles con las operaciones de regresin lineal.

No hay una relacin estadsticamente significativa entre estudiar en una escuela u otra.

Tampoco hay una relacin estadsticamente significativa entre la relacin G2/G1, por lo que me equivoqu al pensar que habra una relacin entre la mejora del primer trmino sobre el segundo.

Hay una relacin con un pvalue menor a 0.05 entre las faltas y la calificacin final, lo que en un inicio no pareciera hacer mucho sentido, pero el coeficiente es muy pequeo, por cada falta se esperara un aumento de 0.05 puntos de 20, por lo que a pesar del p value no creo que sea muy significativa.
